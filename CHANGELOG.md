# Changelog

All notable changes to the CookiePrime Android SDK will be documented in this file.

## [1.0.0] - 2026-06-23
### Added
- Initial public release of the CookiePrime Android SDK.
- On-device dependency scanner capable of detecting 27+ known SDKs and matching patterns for unknown SDKs.
- `CookiePrime.init()` with auto-initialization and 30-day trial support via `TRIAL-12345678`.
- `CookiePrime.showConsentDialog()` for programmatic banner display.
- `CookiePrime.isConsentGiven()` for runtime category checks.
- Native Google Consent Mode v2 and IAB TCF v2.2 compliance implementation.
- Sample integration apps: Sunflower and TicTacToe.
