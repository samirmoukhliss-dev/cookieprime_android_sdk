# Contributing to CookiePrime

Thank you for your interest in the CookiePrime Android SDK! 

Because the core SDK is proprietary software distributed as a pre-compiled `.aar` file, we do not accept pull requests that modify the core engine. However, community feedback is essential to our roadmap.

## How to Contribute

**1. Bug Reports**
If you encounter a runtime crash, a conflict with another library (e.g., Unity Ads, OkHttp), or an undetected tracker, please open an issue using our **Bug Report** template. Include your `adb logcat` output to help us debug faster.

**2. Feature Requests**
Need support for a new analytics platform or an enterprise-specific feature? Open an issue using our **Feature Request** template.

**3. Sample Apps**
We welcome Pull Requests that add new integration examples to the `samples/` directory! If you've integrated CookiePrime into a unique architecture (e.g., Jetpack Compose, KMM), feel free to submit the sample project.
